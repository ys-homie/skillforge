# 多语言同步

## 触发条件
代码实现完成 + 负责人检查自测通过后，如果本次需求有新增多语言词条。

**如果没有新增多语言词条，跳过此 skill。**

## 前提
负责人会在需求开始时提供：
- 翻译管理平台信息（飞书多维表格 / Phrase / Lokalise / 其他）
- 同步的 key 字段和基准语言

## 执行流程

1. 从当前修改的多语言文件中提取**新增**的 key 及对应翻译值
2. 以指定的基准语言为主，连同其他语言列一起整理
3. 按平台 API 格式整理批量数据
4. 执行写入或输出数据供负责人确认

## 注意事项

- 只同步**新增**词条，不要重复写入已有的
- 单次写入量超过平台限制时需分批
- 写入前先确认平台字段结构
- 临时文件用完即删
